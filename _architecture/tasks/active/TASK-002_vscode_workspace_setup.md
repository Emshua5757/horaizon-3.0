## Context

The repo needs a `.vscode/settings.json` so that `toggle_view.py` can focus the workspace on one module at a time without getting overwhelmed by all the sibling folders. This task also sets up workspace-level editor settings (ruler, format on save, file associations) that benefit all coding tasks.
